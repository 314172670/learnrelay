# Nested Queries

In this chapter, we will get in touch with important [terminology](terminology.md) that Relay uses.

We will instill some life in our app and list all the Pokemon we own so far by [putting together all](putting-it-all-together.md) that we learned so far.

As a bonus, we will learn how to use the powerful concept of [cursors](cursors-pagination.md) to easily implement pagination.
